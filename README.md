# Jemarco Briz — Portfolio

Personal portfolio website for Jemarco N. Briz, a Computer Science student at the University of the Philippines Visayas and aspiring Data Engineer.

🔗 **Live site:** 

## Tech Stack

- **Framework:** [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vite.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Routing:** [React Router](https://reactrouter.com/)
- **Fonts:** Schibsted Grotesk · Geist Mono

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
